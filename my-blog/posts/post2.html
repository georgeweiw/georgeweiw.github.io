GitHub Pages 搭建博客教程

2025年11月4日 技术

GitHub Pages 是 GitHub 提供的一项免费静态网站托管服务，可以用于搭建个人博客、项目文档等。

为什么选择 GitHub Pages？

  • 完全免费
  • 无需服务器
  • 自动部署
  • 支持自定义域名
  • 版本控制

搭建步骤

  1. 创建 GitHub 账户
  2. 新建仓库，命名为 username.github.io
  3. 上传你的网站文件
  4. 访问 https://username.github.io

通过这个简单的教程，你可以在几分钟内拥有自己的博客网站！
